Metadatos del archivo
01
FileReader API
El archivo se lee como ArrayBuffer binario en tu navegador. Ningún byte sale de tu máquina.
02
WebCrypto SHA-256
crypto.subtle.digest('SHA-256', buffer) — equivalente exacto a hashlib.sha256() de Python.
03
hexdigest()
El ArrayBuffer resultante se convierte a 64 caracteres hexadecimales — idéntico al output de Python.
04
Comparación
Si los hashes son iguales → integridad garantizada. Si difieren → el archivo fue alterado o corrompido.
